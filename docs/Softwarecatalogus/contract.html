Contract

Een Contract beschrijft de formele overeenkomst voor het inzetten van een VoorzieningAanbod op een VoorzieningGebruik.

  • Specificaties
  • Relaties
id
required
string

Unieke identifier van het Contract

required
object

Het aanbod van de voorziening waarop dit contract betrekking heeft

required
object

Het gebruik van de voorziening waarop dit contract betrekking heeft

startDatum
required
string <date>

De startdatum van het contract

eindDatum
string <date>

De einddatum van het contract (indien van toepassing)

contractNummer
required
string

Het referentienummer van het contract

contractType
required
string
Enum: "SLA" "Licentie" "Onderhoud"

Het type contract

kosten
number

De kosten verbonden aan het contract

kostenPeriode
string
Enum: "Maandelijks" "Jaarlijks" "Eenmalig"

De periode waarop de kosten betrekking hebben

object

De contactpersoon bij de aanbieder

object

De contactpersoon bij de gebruiker

documentReferentie
string

Referentie naar het contractdocument

status
required
string
Enum: "Actief" "Verlopen" "In onderhandeling"

De status van het contract

opmerkingen
string

Aanvullende informatie over het contract
